# Stage 1: Signal Capture

## Narrative

Your personal terminal flickers into life.

AUGUR has locked the city behind a sequence of access checks, but the first barrier is still unstable. Buried inside your terminal feed is a fragmented emergency unlock signal. Most of the stream is noise. Some rows are decoys. Some are real. Hidden inside the surviving signal is a 6-digit PIN tied to your operator identity.

AUGUR has also corrupted the recovery script before leaving it behind. Subtle. Annoying. Very on brand.

Your first job is not to write something from scratch. It is to do what real analysts and developers often have to do under pressure: take a broken script, repair it, run it on messy data, and recover the result.

Fix the syntax error. Process the dataset. Filter the valid signal rows. Decode the digits in the correct order.

This is your first breach point.

Recover your personal 6-digit Stage 1 code and submit it to AUGUR PODIUM.

## Instructions

1. Create a new directory called stage_1 in your VS Code project.
2. Create a new script file in that directory using the filename below for your chosen language.
3. Download the Stage 1 dataset from this page and save it in stage_1 as stage1_dataset.csv.
4. Copy and paste the starter code exactly as shown, then debug syntax errors until it runs.
5. Run the corrected script on stage1_dataset.csv to extract your personal 6-digit stage code.
